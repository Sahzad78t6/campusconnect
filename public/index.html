🚀 About CampusConnect


  

    
 CampusConnect is a full-stack student collaboration platform built to connect students through events, marketplace trading, doubt solving, and structured career roadmaps. 
 
 The platform integrates real-time event countdowns, secure authentication, marketplace payment simulation, admin analytics dashboard, and personalized career progress tracking. 
 
 Built using MERN Stack (MongoDB, Express, Node.js) with scalable architecture and modular backend design.
